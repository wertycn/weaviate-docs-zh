此OIDC流程使用用户名和密码来获取所需的身份验证令牌。

请注意，并非每个提供程序都会自动包含`refresh token`，并且可能需要适当的*scope*，这取决于您的身份提供程序。客户端默认使用*offline_access*作为范围。这适用于某些提供程序，但由于它取决于身份提供程序的配置，我们建议您参考身份提供程序的文档。

如果没有刷新令牌，就无法获取新的`访问令牌`，并且在令牌过期后客户端将变为未认证状态。

:::note
The Weaviate client does not save the username or password used.

They are only used to obtain the first tokens, after which existing tokens will be used to obtain subsequent tokens if possible.
:::
